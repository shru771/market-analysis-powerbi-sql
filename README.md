# 📊 Market Analysis using SQL & Power BI

## 📌 Project Overview

This project analyzes customer purchasing behavior and product performance using **SQL** and **Power BI**.

The objective is to identify customer trends, reorder behavior, top-performing products, and generate business insights that support marketing and inventory decisions.

---

## 🛠️ Tools Used

- 🗄️ MySQL
- 📊 Power BI
- 📑 Microsoft Excel

---

## 📂 Dataset

The project uses five datasets:

1. 🛒 Orders
2. 📦 Products
3. 🏢 Departments
4. 🏬 Aisles
5. 🔄 Order Products Prior

---

## ❓ Business Questions Solved

✅ Top 10 aisles with highest number of products

✅ Product distribution across departments

✅ Top reordered products

✅ Peak ordering hours

✅ Customer retention analysis

✅ Average order size

✅ Department performance

✅ Reorder rate analysis

✅ Customer purchasing behavior

## 🔍 Key Findings

- 🔄 Reorder rate indicates strong customer loyalty.
- 🥬 Produce department contributes the highest product volume.
- ⏰ Peak ordering activity occurs around **10 AM**.
- 📅 Weekend order activity is higher than weekdays.
- 🍌 Frequently reordered products indicate strong customer demand.

---

## 💼 Business Recommendations

- 🎯 Focus promotions during peak hours.
- 📦 Improve inventory planning for top products.
- 🤝 Implement customer loyalty programs.
- 📈 Use reorder trends for personalized marketing.
- 🛒 Maintain stock for high-demand products.
